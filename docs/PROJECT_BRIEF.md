# Project Vision & Impact

We are building a self‑service, AGNO‑powered chatbot that turns raw user complaints into fully reproducible Zendesk tickets in seconds, without human intervention.

**Benefits**
- Cuts average bug‑resolution time ≥ 25 %
- Reduces support back‑and‑forth emails ≥ 70 %
- Produces structured data for future auto‑triage & analytics

**Scope:**  
Frontend widget on Vercel, backend AGNO FastAPI container, Zendesk ticketing.
